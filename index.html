QFA

Quick Font Awesome

Download this project as a .zip file Download this project as a tar.gz file

Provides method for embedding Font Awesome in WordPress.

Future updates (coming in August) will add a TinyMCE 'drawer', as well as a 'always use latest FA CSS' option

NOTE: This is a quick plugin I wrote for my own use; there are a number of items I must put into place before I would consider it acceptable for widespread use. Nearly all of the big, cool stuff I've written is proprietary and so cannot be released to the OSS community
